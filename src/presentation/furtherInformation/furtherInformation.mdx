#### Reminder: Check Out Redux Starter Kit!

Redux Starter Kit is the official "batteries included" toolset to help simplify writing Redux code:

- `configureStore` automatically sets up a store with:
    - `thunk` middleware
    - checks for accidental mutations and non-serializable values in state
    - the Redux DevTools extension (including "action stack traces" tracking )
- `createSlice` simplifies writing Redux logic:
    - Write reducer functions, generate action creators and action types automatically
    - Reducers use Immer to let you write "mutative" immutable updates
    - Makes it easy to write single-file "ducks"-style code structure
- Written in TypeScript, works great with TS and JS apps
- Useful for all Redux users of all experience levels
